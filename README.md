# System-Management & Web-GUI Tool (in Arbeit)

Ein modulares **Python-basiertes Softwareprojekt** mit einer grafischen Benutzeroberfläche (GUI) zur Demonstration von objektorientierter Programmierung (OOP) und sauberer Code-Architektur.

## 🏗️ Projektstruktur & Architektur
Das Projekt folgt dem Prinzip der strikten Trennung von Zuständigkeiten (Separation of Concerns):
- `core/`: Enthält die Kernlogik und Geschäftslogik der Anwendung.
- `gui/`: Verwaltet die grafische Benutzeroberfläche und Benutzerinteraktionen.
- `config/`: Speichert globale Einstellungen und Systemkonfigurationen.
- `utils/`: Hilfsfunktionen und wiederverwendbare Skripte.
- `data/`: Lokale Datenspeicherung und Protokolle.

## 🛠️ Technologien & Konzepte
- **Programmiersprache:** Python 100%
- **Architektur:** Modulares Design, OOP (Objektorientierte Programmierung)
- **Konfigurationsmanagement:** Zentralisierte Verwaltung von App-Einstellungen
